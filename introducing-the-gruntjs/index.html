GruntJS

Uma breve introdução à ferramenta {que é boa uma barbaridade}

O que é o GruntJS afinal?

  • Ferramenta escrita em NodeJS que gerencia procedimentos do work-flow de desenvolvimento
  • Escrita por Ben Alman - @cowboy
  • Lançada na BrazilJS de 2012 ,aqui em POA mesmo
  • Nasceu de uma necessidade de build no time desenvolvimento do jQuery
  • É extensível através de plug-ins, que estão todos disponíveis no NPM

Alguém deve ter se perguntado...

...escrita em NodeJS?

NodeJS não é uma técnologia de servidor?

NodeJS é muito popular por rodar no servidor. Porém ela possui outras aplicações, como programas CLI. Assim sendo, ele é instalado apenas na máquina de desenvolvimento.

Facilidade de distribuição do GruntJS

Apenas dois arquivos precisam estar no repositório de versionamento.

package.json e Gruntfile.js

Para começar

Após instalarmos o NodeJS...

$ sudo npm install -g grunt-cli

package.json

Responsável pelo gerenciamento das dependências (plugins)

Após isso

$ sudo npm install

Para instalar as dependências descritas no package.json

Gruntfile.js

Responsável pela configuração de todos os plugins e tarefas
